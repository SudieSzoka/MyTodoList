# 使用说明

## 基本使用

### 1. 启动应用

直接打开 `index.html` 文件即可使用，或者使用本地服务器：

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000
```

然后访问 `http://localhost:8000`

### 2. 添加任务

1. 在输入框中输入任务标题
2. 选择优先级（高、中、低）
3. 可选：添加备注描述
4. 点击"添加"按钮

### 3. 管理任务

- **完成任务**：点击 ✅ 按钮
- **恢复任务**：点击 ↩️ 按钮
- **编辑任务**：点击 ✏️ 按钮
- **删除任务**：点击 🗑️ 按钮
- **添加子任务**：点击 ➕ 按钮

### 4. 查看任务

- **进行中**：显示未完成的任务
- **已完成**：显示已完成的任务

## GitHub 同步

### 同步步骤

1. 点击"同步到 GitHub"按钮
2. 在弹出的对话框中选择：
   - **下载 todo.json**：下载文件到本地
   - **复制数据**：复制数据到剪贴板
3. 打开 GitHub 仓库
4. 编辑 `todo.json` 文件
5. 粘贴数据并保存
6. GitHub Action 会自动触发并同步数据

## 数据备份

- **本地备份**：数据自动保存在浏览器 localStorage 中
- **GitHub 备份**：通过 GitHub Actions 自动备份

## 快捷键

- `Enter`：快速添加任务（在输入框中）
- `Escape`：取消编辑（在编辑模式下）

## 故障排除

### 无法加载数据

1. 检查 `todo.json` 文件是否存在
2. 确认文件格式正确（JSON 格式）
3. 检查浏览器控制台是否有错误信息

### 同步失败

1. 检查网络连接
2. 确认 GitHub 仓库权限
3. 查看 GitHub Actions 运行状态
